Metrics Explanation

Weighted F1 Score
Harmonic mean of precision and recall, weighted by class support. Ranges from 0 to 1 (higher is better).
Macro F1 Score
Unweighted average F1 across all classes. Useful for evaluating performance on minority classes.
Accuracy
Percentage of correctly classified games. Simple overall performance metric.

Note: The ensemble model combines predictions from Random Forest, Gradient Boosting, and XGBoost using a stacked meta-learner (XGBoost Classifier). This approach leverages the strengths of each base model to achieve superior generalization.
