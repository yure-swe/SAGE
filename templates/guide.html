🎯 Understanding Results

Predicted Owner Tier

The model classifies your game into one of 6 ordinal owner tiers:

  • Class 0 (≤10K): Common Indie — Up to 10,000 owners
  • Class 1 (35K): Niche — 10,000 – 35,000 owners
  • Class 2 (75K): Growing — 35,000 – 75,000 owners
  • Class 3 (150K): Established — 75,000 – 150,000 owners
  • Class 4 (350K): Popular — 150,000 – 350,000 owners
  • Class 5 (≥750K): Breakout Hit — 750,000+ owners

Confidence Score

The confidence percentage (0-100%) represents how certain the model is about its prediction. Higher confidence indicates stronger signal from the features.

  • 80-100%: Very confident prediction
  • 60-79%: Moderately confident
  • 40-59%: Low confidence (borderline case)
  • Below 40%: Uncertain (features are mixed/weak)

Class Probabilities

Shows the predicted probability for each of the 6 classes. The class with the highest probability is selected as the final prediction.

Recommendations

  • Strengths: Features currently working in your favor (high positive impact)
  • Improvements: Actionable suggestions to increase predicted success tier (based on SHAP analysis)

💡 Best Practices

  • Be Accurate: Enter realistic values. The model is trained on real Steam data, so inflated numbers may lead to misleading predictions.
  • Focus on Pre-Launch Features: This model predicts success based ONLY on pre-launch attributes (no reviews, no playtime data). Only fill in features you can control before release.
  • Use Recommendations: The improvement suggestions are based on SHAP feature importance. Prioritize high-impact features (higher impact scores).
  • Test Multiple Scenarios: Try different configurations (e.g., with/without publisher, different pricing) to see how predictions change.
  • Don't Over-Optimize: The model shows correlations, not causation. A high prediction doesn't guarantee success—focus on game quality first.

❓ Frequently Asked Questions

What if I don't have a publisher?

Set has_publisher to 0 (unchecked) and publisher_count to 0. The model will adjust predictions accordingly. Note that publisher backing is a positive signal but not required for success.

Can I predict for unreleased games?

Yes! This model is specifically designed for pre-launch prediction. All 48 features are available before release (pricing, store page, platform support, etc.).

What if some features are unknown?

Use the default values provided in the form. For binary features (yes/no toggles), leave unchecked (0) if the feature is not planned.

Why is my prediction confidence low?

Low confidence occurs when feature values fall in "borderline" regions where multiple classes overlap. Try adjusting key features (store page quality, pricing, platform support) to see if confidence improves.

How accurate is the model?

The ensemble achieves 70.85% accuracy and 0.6211 weighted F1 score on the test set. See the Model Metrics page for detailed performance.

Can I use this for platforms other than Steam?

No. The model is trained exclusively on Steam data. Predictions for Epic Games, GOG, or console platforms may not be accurate.
